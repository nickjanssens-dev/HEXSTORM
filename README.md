# HEXSTORM 🔥

Hexstorm is a pseudo-3D raycasting action game. 
Players take control of a battle mage and fight enemies using a combination of magical projectiles, laser-based attacks, and real-world interaction through cards.

---

## 🎮 Features

- Pseudo-3D rendering using raycasting 
- First-person movement and combat
- Multiple weapon types:
  - Projectile spells (fireballs, magic bolts)
  - Hitscan attacks (laser / instant shots)
- Card-based spell casting (via webcam)
- Enemy AI and wave-based combat (planned)

---

## 🛠️ Technologies Used

- Python 3.13
- Pygame (game engine)

---

## 🎯 Controls

- ⬆️ / ⬇️ → Move forward / backward  
- ⬅️ / ➡️ → Strafe left / right  
- S / D → Rotate camera  
- (Mouse / Space) → Shoot *(to be implemented)*  
- (Card) → Cast spells *(planned)*  

---

## 🧠 Project Goal

The goal of this project is to combine:
- classic game development techniques (raycasting)
- modern computer vision (card input)

to create an interactive and immersive gameplay experience.
